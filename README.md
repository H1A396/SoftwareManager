# 软件管理器 (Software Manager)

一个基于 Electron 开发的桌面应用程序，用于管理和组织您的软件安装包、下载链接及相关文档。

## 📖 项目简介

软件管理器是一个功能强大的桌面应用程序，帮助您：
- 管理本地安装包和在线下载链接
- 组织和分类您的软件收藏
- 快速查找和访问软件
- 记录安装说明和文档
- 一键打开文件位置或下载链接

---

## ✨ 核心功能

### 1. 📦 软件管理

#### 添加软件
- **软件名称**：必填，软件的名称
- **版本号**：可选，软件版本信息
- **安装方式**：
  - 在线安装：保存下载链接
  - 本地安装：选择本地安装包文件
- **安装包位置/下载链接**：必填，文件路径或URL
- **分类**：可选，支持多种分类（开发工具、办公软件、媒体工具、系统工具、浏览器、其他）
- **安装说明**：可选，文本形式的安装步骤或注意事项
- **MD文档说明**：可选，支持Markdown格式的详细文档

#### 编辑软件
- 支持编辑所有软件信息
- 无需删除重新添加

#### 删除软件
- **单个删除**：删除指定软件
- **批量删除**：多选后批量删除
- **删除所有**：一键清空所有软件

---

### 2. 🔍 搜索与筛选

#### 搜索功能
- **实时搜索**：输入即搜索
- **多字段搜索**：
  - 软件名称
  - 分类
  - 版本号
  - 安装方式（支持中英文搜索）
  - 标签

#### 筛选功能
- **分类筛选**：按软件分类筛选
- **安装方式筛选**：在线安装 / 本地安装
- **标签筛选**：按标签筛选软件
- **状态筛选**：已安装 / 未安装
- **收藏筛选**：已收藏 / 未收藏
- **组合筛选**：支持多个筛选条件同时生效

#### 排序功能
- **默认排序**：按添加顺序
- **收藏优先**：收藏的软件排在前面
- **名称排序**：A-Z / Z-A
- **时间排序**：最新添加 / 最旧添加
- **分类排序**：按分类名称
- **安装方式排序**：按安装方式
- **评分排序**：高-低 / 低-高

---

### 3. 🏷️ 标签管理

#### 标签功能
- **添加标签**：在添加或编辑软件时添加标签
- **标签管理窗口**：单独的标签管理界面
- **搜索标签**：在标签管理窗口中搜索标签
- **标签选择**：从现有标签中选择或输入新标签
- **点击筛选**：点击软件卡片上的标签可快速筛选

#### 标签筛选下拉菜单
- 显示所有使用过的标签
- 标签按字母顺序排序
- 选择标签后自动筛选软件
- 数据变化时自动更新标签列表

---

### 4. 📋 文档管理

#### 文本说明
- 支持文本形式的安装说明
- 支持换行和基本格式

#### Markdown文档
- 支持添加外部.md文件
- 实时渲染Markdown格式
- 支持修改MD文件路径
- 独立窗口查看MD文档
- 支持滚动浏览长文档

### 6. 🎨 主题与界面

#### 主题切换
- **亮色模式**：清爽明亮的白色背景
- **暗色模式**：护眼舒适的深色背景
- **一键切换**：右上角月亮/太阳图标快速切换

#### 界面美化
- **个性化滚动条**：紫色渐变圆角滚动条
- **卡片式设计**：现代化卡片布局
- **渐变色背景**：优雅的渐变效果
- **平滑动画**：流畅的交互动画
- **响应式布局**：自适应不同窗口大小

### 7. 💾 数据管理

#### 自动保存
- 添加、编辑、删除操作自动保存
- 数据存储在软件所在目录的 `config` 文件夹中

#### 导出数据
- 导出为JSON格式
- 默认保存到 `config` 文件夹
- 可自定义保存位置

#### 导入数据
- 支持导入JSON格式的数据文件
- 默认打开 `config` 文件夹
- 可选择其他位置的数据文件

## 📦 安装与运行

### 开发环境

#### 1. 克隆项目
```bash
git clone <项目地址>
cd SoftwareManager
```

#### 2. 安装依赖
```bash
npm install
```

#### 3. 运行项目
```bash
npm start
```

### 生产环境

#### 打包应用
```bash
# 打包为Windows安装包

# 打包后的文件在 dist 目录中
```
