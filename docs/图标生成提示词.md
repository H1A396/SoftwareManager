# 软件管理器图标生成提示词

## 提示词内容

### 通用要求
请为"软件管理器"应用设计一套专业、现代的图标，包括应用图标和安装包图标。

### 应用图标设计要求

**设计风格：**
- 现代扁平化设计，符合 Windows 11 设计规范
- 简洁、专业、易于识别
- 使用圆角设计，亲和力强
- 视觉平衡，在小尺寸下清晰可读

**核心元素：**
- 主要隐喻：软件包/应用图标 + 管理工具的组合
- 可以使用：文件夹、盒子、工具箱、齿轮、列表等元素的组合
- 建议使用2-3个核心元素，不要过于复杂

**颜色方案：**
- 主色调：蓝色系（#667eea 或类似的专业蓝色）
- 辅助色：白色、浅灰色
- 渐变：从左上到右下的微妙渐变，角度约120度
- 避免过多颜色，保持简洁

**尺寸要求：**
- 标准尺寸：256x256 像素
- 支持缩放：16x16, 32x32, 48x48, 64x64, 128x128, 256x256
- 48x48 网格对齐，圆角半径：外部曲线 2px，内部曲线 1px

**具体设计建议：**
1. **方案一**：一个现代化的软件盒子/文件夹图标，带有管理齿轮或列表元素
2. **方案二**：一个工具箱图标，内部包含多个小应用图标
3. **方案三**：一个抽象的软件管理界面图标，包含网格或列表元素

### 安装包图标设计要求

**设计风格：**
- 与应用图标保持一致的设计语言
- 可以稍微复杂一些，因为显示尺寸较大
- 包含安装相关的视觉元素

**核心元素：**
- 在应用图标基础上添加安装相关元素
- 可以添加：向下箭头、光盘、安装进度条、加号等
- 保持与应用图标的视觉连贯性

**颜色方案：**
- 与应用图标使用相同的颜色方案
- 可以添加绿色元素表示"安装"或"开始"

**尺寸要求：**
- 主要尺寸：256x256 像素
- 用于安装程序和桌面快捷方式

### 技术要求

**输出格式：**
- PNG 格式（透明背景）
- ICO 格式（用于 Windows 应用图标）
- SVG 格式（矢量图，便于缩放和修改）

**分辨率：**
- 高分辨率版本：512x512 像素
- 标准版本：256x256 像素
- 小尺寸版本：32x32, 48x48 像素

**文件命名：**
- app-icon.png - 应用图标
- app-icon.ico - Windows 应用图标
- installer-icon.png - 安装包图标
- app-icon.svg - 矢量版本

### 设计参考

**参考风格：**
- Microsoft Office 图标风格
- Windows 11 系统应用图标
- 现代扁平化设计趋势

**避免的设计：**
- 过于复杂的细节
- 文字或字母
- 过多的颜色和渐变
- 尖锐的角和边

### 使用场景

**应用图标用于：**
- 桌面快捷方式
- 任务栏图标
- 开始菜单
- 应用窗口标题栏
- 安装程序

**安装包图标用于：**
- 安装程序文件
- 卸载程序
- 控制面板中的程序列表

### 交付要求

请提供：
1. 至少 3 个不同设计方案供选择
2. 每个方案包含应用图标和安装包图标
3. 多种尺寸和格式的输出文件
4. 设计说明和设计思路
5. 可编辑的源文件（如 SVG 或设计工具文件）

### 额外说明

这个软件管理器的主要功能是：
- 管理本地安装的软件
- 记录软件的安装路径、下载链接、版本信息
- 支持软件的添加、编辑、删除
- 支持批量操作
- 支持数据导入导出

图标应该体现"管理"、"组织"、"工具"等概念，同时保持专业和现代的感觉。
