# 豆包图标生成提示词（简化版）

## 直接复制使用

请为"软件管理器"应用设计一套专业、现代的图标，包括应用图标和安装包图标。

## 设计要求

### 应用图标
- **风格**：现代扁平化，符合 Windows 11 设计规范
- **核心元素**：软件包/文件夹 + 管理工具（齿轮、列表等）的组合
- **颜色**：蓝色系主色调（#667eea），白色和浅灰色辅助，微妙渐变
- **尺寸**：256x256 像素，支持缩放到 16x16, 32x32, 48x48, 64x64, 128x128
- **设计**：圆角设计，亲和力强，小尺寸下清晰可读

### 安装包图标
- **风格**：与应用图标保持一致
- **额外元素**：添加安装相关元素（向下箭头、加号等）
- **颜色**：相同配色，可添加绿色表示"安装"
- **尺寸**：256x256 像素

### 输出格式
- PNG 格式（透明背景）
- ICO 格式（Windows 应用图标）
- SVG 格式（矢量图）
- 文件命名：app-icon.png, app-icon.ico, installer-icon.png, app-icon.svg

### 设计建议
1. **方案一**：现代化软件盒子 + 管理齿轮
2. **方案二**：工具箱 + 多个小应用图标
3. **方案三**：抽象管理界面 + 网格/列表元素

### 参考风格
- Microsoft Office 图标风格
- Windows 11 系统应用图标
- 现代扁平化设计

### 软件功能
- 管理本地安装的软件
- 记录软件路径、下载链接、版本信息
- 支持添加、编辑、删除、批量操作
- 支持数据导入导出

图标应体现"管理"、"组织"、"工具"概念，保持专业和现代感。

请提供至少 3 个设计方案供选择。
