# 软件管理器 - 打包和分发指南

## 打包步骤

### 1. 打包应用

在项目根目录下运行以下命令：

```bash
npm run build
```

这会在 `dist` 文件夹中生成以下文件：

- **软件管理器 Setup 1.0.0.exe** - Windows 安装程序（推荐）
- **软件管理器 1.0.0.exe** - 便携版（无需安装，直接运行）

### 2. 分发方式

#### 方式一：安装程序（推荐）
- 分发 `dist/软件管理器 Setup 1.0.0.exe`
- 用户双击安装，会创建桌面快捷方式和开始菜单项
- 可以选择安装目录

#### 方式二：便携版
- 分发 `dist/软件管理器 1.0.0.exe`
- 用户无需安装，直接双击运行
- 不会在系统中留下痕迹

## 打包配置说明

当前配置会生成两种格式的安装包：

1. **NSIS 安装程序**：传统的 Windows 安装程序
   - 支持自定义安装目录
   - 创建桌面快捷方式
   - 创建开始菜单快捷方式

2. **便携版**：单文件可执行程序
   - 无需安装
   - 直接运行

## 自定义配置

### 修改应用名称

编辑 `package.json` 文件中的 `productName` 字段：

```json
{
  "productName": "你的应用名称"
}
```

### 修改版本号

编辑 `package.json` 文件中的 `version` 字段：

```json
{
  "version": "1.0.1"
}
```

### 添加自定义图标（可选）

1. 准备一个 `.ico` 格式的图标文件
2. 将图标文件命名为 `icon.ico`
3. 放在项目根目录
4. 修改 `package.json` 添加图标配置：

```json
{
  "build": {
    "win": {
      "target": ["nsis", "portable"],
      "icon": "icon.ico"
    }
  }
}
```

5. 修改 `main.js` 添加图标配置：

```javascript
function createWindow() {
    mainWindow = new BrowserWindow({
        // ... 其他配置
        icon: path.join(__dirname, 'icon.ico'),
        title: '软件管理器'
    });
}
```

## 注意事项

1. **首次打包**：首次打包可能需要下载 Electron 二进制文件，会比较慢
2. **文件大小**：打包后的文件较大（约 100-200MB），这是正常的
3. **分发建议**：建议使用安装程序，用户体验更好
4. **测试**：打包后先自己测试安装和运行，确保功能正常

## 给用户使用

### 安装程序使用方式

1. 下载 `软件管理器 Setup 1.0.0.exe`
2. 双击运行安装程序
3. 按照提示完成安装
4. 从桌面或开始菜单启动应用

### 便携版使用方式

1. 下载 `软件管理器 1.0.0.exe`
2. 放在任意文件夹中
3. 双击运行即可

## 更新应用

修改代码后，重新运行打包命令即可生成新版本：

```bash
npm run build
```

记得在 `package.json` 中更新版本号。
